# E-Commerce CMS

E-Commerce CMS is an server-side / admin-side application to manage E-Commerce client-side. This app has:
* RESTful endpoint for Product CRUD Operation
* Used Technology : 
    - Express Js, 
    - Sequelize, 
    - Postgres, 
    - Jest (JS Testing Framework)
    - Json Web Token, 
    - Bcrypt
* JSON Formated Response
